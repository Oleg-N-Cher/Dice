This little game demonstrates how to develop
a game for ZX Spectrum in Oberon language.

Эта маленькая игра демонстрирует кроссплатформенный
подход XDev при разработке на языке Оберон. Сейчас готова
версия для ZX Spectrum. Планируются целевые платформы:

Java Micro Edition
Linux x86-32
MS DOS 16 bit
Sdl-Linux x86-32
Sdl-Windows 32
Sdl-Windows 64
Windows 32
Windows 64
ZX Spectrum 48k, 128k

Хочу попутно заметить, что обычно при работе на каком-то
средстве разработки рано или поздно сталкиваешься с какими-то
непреодолимыми ограничениями. В рамках парадигмы XDev
программист волен вполне успешно двигаться практически
в любом направлении, для любой платформы.

Уверен, что размеченная схема наглядно показывает потенциал
подхода XDev. Соль здесь не в том, что так можно разрабатывать
демки. Соль в максимальном выводе алгоритма (естественно,
путём его продумывания) из зоны низкого уровня, что с выбором
удачных межплатформенных абстракций позволяет начать накопление
алгоритмической базы исходников на Обероне, и по мере её
нарастания будет сохраняться возможность использовать их
на любой платформе, как имеющейся, так и появившейся в будущем.

Инструкция по сборке.
